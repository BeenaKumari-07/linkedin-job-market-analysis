# 📊 LinkedIn Job Market Trends Analysis

An end-to-end Python data pipeline that scrapes live job listings to analyze hiring patterns, top locations, and in-demand titles for Data Analysts.

## 🚀 Overview
This project automates the extraction of job market data. By transforming raw web data into structured insights, it identifies which cities and companies are currently leading the market for Data Science roles.

## 🛠️ Tech Stack
- **Languages:** Python
- **Scraping:** BeautifulSoup, Requests
- **Data Wrangling:** Pandas, NumPy
- **Visualization:** Seaborn, Matplotlib

## 🔍 Key Insights
- **Geographic Density:** Mapped the top 10 hiring hubs for Analysts.
- **Role Distribution:** Categorized job titles to find the most common variations of "Data Analyst."
- **Market Analysis:** Used Heatmaps to visualize the relationship between Job Titles and Locations.
